Tajniacy – Console Edition (C#)

Overview
A simple console-based version of the board game Tajniacy (Codenames), created in C# as an experiment to test communication between the game logic and Large Language Models (LLMs).

Features

Console interface with clear text output

Automatic word grid generation

Support for multiple teams (Red vs Blue)

Clue–guess turn logic

Easy-to-modify codebase for experiments with AI agents

Requirements

.NET 6.0 or newer

Visual Studio or VS Code

Works on Windows, macOS, and Linux terminal

How to Run

Clone the repository:
git clone https://github.com/WZIMNIACY/Tajnicy-basic-console-version-C-.git

Enter the project directory:
cd Tajnicy-basic-console-version-C-/ConsoleApp1

Run the project:
dotnet run

Project Structure
ConsoleApp1/ → main game logic
Notatki/ → developer notes and ideas
Testy/ → test files and experimental features
ConsoleApp1.sln → Visual Studio solution file
